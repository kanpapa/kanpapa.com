VAIO type PでWindows 7 RCを試用していますが、なかなかいい調子です。 すでにアップデートも何度かかかりました。更新履歴を確認すると、

- Microsoft driver update for Atheros AR928X Wireless Network Adapter
- Update for Windows 7 Release Candidate (KB970789)
- Update for Internet Explorer 8 Compatibility View List for Windows 7 Release Candidate (KB969497)
- Update for Internet Explorer 8 for Windows 7 Release Candidate (KB970858)
- Definition Update for Windows Defender - KB915597 (Definition 1.57.806.0)
- Intel Corporation driver update for Intel(R) Graphics Media Accelerator 500
- Sony Corporation driver update for Sony Programmable I/O Control Device

といった具合です。SonyのPIOデバイスとか、GMA500のアップデートとかも適用されてます。 この調子でどんどんアップデートしてチューニングしてほしいですね。
