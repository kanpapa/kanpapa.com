まずログインは早いです。すぐデスクトップが表示されるようになりました。

あとはドックの奇麗さ。ドックは鏡の上にアプリケーションが置いてあるようなデザインですが、この鏡にアプリケーションが反射されて表示され、おまけにウィンドウを重ねると、その内容も鏡に映っているように見えるのです。この点は奇麗ですね。

ドックに映っているウィンドウがわかりますか？

![leopard_dock_20071028081539.jpg](images/leopard_dock_20071028081539.jpg)

あと、我が家で使っているアプリケーションの動作状況です。

- KORG M1-Le → OK
- KORG K25のドライバ → OK
- ToyViewer → OK
- SmartLogBook → OK
- NeoOffice → OK
- BOINC Manager (再インストールしたら動作しました。） → OK
- Camino → OK
- Parallels Desktop → OK?（ただしレジューム状態にすると次回起動がおかしいときあり）
- Google Earth → OK
- PasswordMaster3 → NG

Boinc Managerはパーミッションがうんぬんというエラーがでましたが、再度インストールしたら問題なくなりました。

PasswordMaster3はファイルを読みにいったまま先に進まない状態になります。

追記：[PasswordMaster3の作者のページ](http://hp.vector.co.jp/authors/VA024217/)では今後マックユーザ向けパーソナルセキュリティソリューション「iSecUtilities」を2007年10月下旬にも提供とのことですので、そちらを使わせて頂こうと思います。
