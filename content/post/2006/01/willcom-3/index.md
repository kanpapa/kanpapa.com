ようやく自宅に帰り着きました。ポストの中をみるとWILLCOMからのご利用料金のお知らせが届いていました。

京ぽん２になってからの初めての請求です。偶然にもこの月からA&B割の適用になりましたので、料金がどうなっているかなと明細を見たところ、

| パケコミネット基本使用料 | 4,700 |
| --- | --- |
| 長期割引額 | \-235 |
| 年間契約割引額 | \-705 |
| 複数割引 | \-200 |
| A&B割割引額 | \-1,410 |
| 通話料 | 60 |
| 高速化サービス月額料 | 300 |
| 電話機変更手数料 | 2,000 |
| 小計 | 4,510 |

え?と、電話機変更手数料の2,000円を差し引くと。なんと2,510円！！！！

WILLCOMさん、本当にいいのでしょうか。こんなに安くて。

確かにパケコミの上限の20万パケットには届かなかったけど。4Xで十分京ぽん２で堪能できているんですが。本当に。

今年もよろしくおねがいします！＞WILLCOMさん
