これまで使ってきたPostToTwitterプラグインを、新しいPostTweetプラグインに入れ替えました。 というのもTwitterがOAuth認証のみになったので、旧プラグインが使えなくなってしまったのです。 PostTweetプラグインは、MovableTypeのプラグインディレクトリに掲載されていますので、そこからたどってダウンロードします。 インストール自体はいつものように解凍したプラグインディレクトリをそのままpluginディレクトリに入れるだけですが、私の場合は設定画面でCrypt::SSLeayが無いと怒られてしまったので、yumでperl-Crypt-SSLeayを追加インストールしました。 $ sudo yum install perl-Crypt-SSLeay Loaded plugins: fastestmirror Loading mirror speeds from cached hostfile \* addons: ftp.iij.ad.jp \* base: ftp.iij.ad.jp \* extras: ftp.iij.ad.jp \* updates: ftp.iij.ad.jp Setting up Install Process Resolving Dependencies --> Running transaction check ---> Package perl-Crypt-SSLeay.x86\_64 0:0.51-11.el5 set to be updated --> Finished Dependency Resolution Dependencies Resolved ================================================================================ Package Arch Version Repository Size ================================================================================ Installing: perl-Crypt-SSLeay x86\_64 0.51-11.el5 base 45 k Transaction Summary ================================================================================ Install 1 Package(s) Upgrade 0 Package(s) Total download size: 45 k Is this ok \[y/N\]: Y Downloading Packages: perl-Crypt-SSLeay-0.51-11.el5.x86\_64.rpm | 45 kB 00:00 Running rpm\_check\_debug Running Transaction Test Finished Transaction Test Transaction Test Succeeded Running Transaction Installing : perl-Crypt-SSLeay 1/1 Installed: perl-Crypt-SSLeay.x86\_64 0:0.51-11.el5 Complete! $ WebにあるインストールマニュアルにOAuthの設定など非常に親切に書かれているので初心者のかたでもOAuth対応ができると思います。ありがたく使わせていただきます。
